Final project for ENGR P132

Amazon and Walmart web scrapper used to compare prices and find the cheapest product.

# How to use
1. Copy code to compiler(VSCode)
2. Clear google chrome cookies
3. Run file name in terminal
   (python price_compare.py)
4. Input name of item you want to search for
5. Terminal should return cheapest items with ratings and names and URL
